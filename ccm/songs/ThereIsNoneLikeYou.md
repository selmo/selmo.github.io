# 주님과 같이 (There Is None Like You)

- **원곡**: There Is None Like You
- **작곡**: Lenny LeBlanc
- **번역 수록**: 야훼이레 #752
- **저작권**: CopyCare

## 개요

미국 CCM 작곡가 Lenny LeBlanc의 대표곡으로, "주님과 같이"라는 제목으로 야훼이레에 수록되었습니다. "주님 같은 분은 어디에도 없다"는 유일신 신앙 고백을 단순한 멜로디 위에 담아냅니다.

## 가사 주제

하느님의 비교 불가능한 유일성과 위대함을 찬양하는 곡입니다. 구약의 쉐마(신명 6:4) 전통과 시편의 찬양 언어를 현대적으로 표현했으며, 장례·위령 미사에서도 자주 선택됩니다.
