# 떼제 공동체 (Taizé Community)

**떼제 공동체**는 프랑스 떼제에 있는 에큐메니칼 수도 공동체입니다. 공동 기도, 성경 묵상, 화해와 일치의 영성을 중심으로 전 세계 젊은이들을 초대해 왔습니다.

## 전례 음악과의 관련성

*   짧은 후렴을 반복하는 떼제 노래는 회중 참여와 묵상적 기도를 결합한 현대 교회음악의 대표적 양식입니다.
*   라틴어와 여러 현대 언어를 함께 사용해 국제 공동체 전례에 잘 어울립니다.
*   미사 통상문 전체를 하나의 대형 세트로 구성하기보다, Kyrie, Sanctus, Agnus Dei처럼 특정 파트를 떼제식으로 사용하는 경우가 많습니다.

## 볼 때 체크할 점

*   떼제 노래는 화려한 진행보다 반복과 정지감이 중요하므로, 빠른 템포의 청년 미사와는 결이 다를 수 있습니다.
*   본당 적용 시에는 묵상 시간, 성체조배, 피정, 국제 공동체 미사 같은 맥락에서 특히 강점을 보입니다.
