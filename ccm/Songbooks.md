# 어린이·청소년·청년 성가집 비교

한국 및 해외에서 사용되는 어린이, 청소년, 청년 대상 가톨릭 성가집과 미사 경본을 연령대별, 출판사별로 정리한 비교 페이지입니다.

## 한국 가톨릭 성가집

### 하늘바다 — 가톨릭 어린이 찬양집

어린이 미사와 주일학교를 위한 독립된 찬양집입니다. 미사 통상문 각 파트에 2-7개씩 어린이용 멜로디를 제공하고, 전례 시기별·주제별 곡을 폭넓게 수록하고 있습니다.

- **구성**: 수록곡마다 고유 번호 부여, 작곡·편곡자 표기.
- **실제 수록곡 목록**: `resources/하늘바다.md`

**미사곡 파트** (각 파트별 복수 설정):

| 파트 | 설정 수 | 주요 작곡·편곡자 |
|:---|:---|:---|
| 성호경 | 4 | 신지은·이지혜, 김 안나, 임두빈, 장환진 |
| 자비송 | 6 | 김종성, 신지은·이지혜, 신상욱, 이종철, 임두빈, 현정수 |
| 대영광송 | 7 | 김종성, 신상욱, 신지은·이지혜, 임두빈, 현정수, 이종철, 임덕 |
| 화답송 | 4 | 신지은·이지혜, 이종철, 박기현, 임두빈 |
| 복음환호송 | 9 | 김종성, 신상욱, 임두빈, 오광설, 현정수, 신지은·이지혜, 그레고리안, 이종철 |
| 사도신경 | 1 | 임두빈 |
| 거룩하시도다 | 5 | 이종철, 신상욱, 김경은, 이승일, 임두빈 |
| 거양성체 | 1 | 신지은·이지혜 |
| 신앙의 신비여 | 2 | 이지혜, 임두빈 |
| 아멘 | 5 | 신상욱, 이종철, 임두빈, 유현주, 신지은·이지혜 |
| 주님의 기도 | 3 | 이종철, 임두빈, 신지은·이지혜 |
| 하느님의 어린양 | 4 | 임두빈, 현정수, 신지은·이지혜, 이종철 |

**전례 시기별 곡**:
- 대림, 성탄, 사순, 부활, 성모, 봉헌, 성체 — 시기별로 수십 곡 수록.

**주제별 분류** (47개 주제):
감사, 기도, 기쁨, 믿음, 사랑, 성령, 소망, 위로, 일치, 찬미·찬양, 친구·나눔, 평화, 하느님의 사랑 등. 주제별로 어린이 눈높이에 맞춘 곡을 찾을 수 있도록 구성.

**성경색인**: 창세기~요한묵시록까지 연관 성경 구절별 곡 색인 제공.

### 야훼이레 (3판) — 수원교구 생활성가집

수원교구에서 발행한 생활성가 모음집으로, 현재 3판까지 출간되었습니다. 미사곡, 화답송, 번역 CCM, 한국 창작 생활성가를 아우르며 사실상 여러 교구의 청소년·청년 미사 공통 레퍼토리 역할을 합니다.

- **수록 규모**: 약 631곡 (3판 기준)
- **실제 수록곡 목록**: `resources/야훼이레-수록곡목록-3판.xlsx` (총괄 시트: 번호, 제목, 작곡자, 저작권자, 원어 정보 포함)
- **부가 정보**: 개신교↔가톨릭 용어 대조표, 가나다순 색인, 성경 구절별 색인 포함.

**곡 구성 (주요 카테고리)**:

| 구분 | 곡 수 | 비고 |
|:---|:---|:---|
| 미사 통상문 (자비송, 대영광송, 거룩하시도다 등) | 다수 | 파트별 복수 작곡가 버전 |
| 화답송 | 다수 | 시편 기반. 연중·대림·사순·부활 시기 구분 |
| 생활성가 (CCM) | 주류 | 한국 창작 + 번역 CCM |
| 성탄 | 11 | |
| 부활 | 9 | |
| 대림 | 9 | |
| 사순 | 8 | |
| 성령 | 12 | 성령 강림·성령 쇄신 주제 |
| 성심 | 3 | |

**특징**:
- 한국 창작곡(최태형, 김정식, 신상옥, 현정수, 이형진, 김태진 등)과 번역 CCM(Bob Fitts, Don Moen, Chris Bowater, Geoff Bullock, Lenny LeBlanc 등)을 함께 수록.
- 미사곡 파트는 작곡가별로 여러 설정을 제공해 본당 밴드 편성과 회중 수준에 맞춰 선택 가능.
- 수원교구 발행이지만 타 교구 청년 미사·피정·찬양 기도회에서도 폭넓게 사용.

## 선택 시 고려할 점

- **전례 일치**: 성가집 선택 시 전례문(미사 통상문 및 고유문 번역)이 승인된 텍스트와 일치하는지 확인. OCP/GIA의 경우 2011년 새 미사경본 번역 대응 여부 체크.
- **회중 난이도**: 어린이와 청소년은 음역이 좁고 반복 구조가 있는 곡이 적합. 청년은 음악적 다양성이 확보된 레퍼토리가 유리.
- **편성**: 밴드(드럼, 기타, 키보드) 중심인지 오르간/피아노 중심인지에 따라 성가집 선택이 달라짐.
- **언어**: 야훼이레는 한국어 번역 CCM을 포함해 국내 청년들이 접근하기 좋음. 반면 *Spirit & Song*은 영어 원어 사용이 일반적.
- **연간판 vs 정본**: 일부 북미 출판사 성가집은 매년 개정되는 연간판(예: *Breaking Bread*)과 고정 합본판(예: *Gather*)이 구분되므로 구매·채택 시 확인 필요.
